1.序列类型

  • 列表：list

    • 列表是一种有序的集合，可以随时添加和删除其中的元素。
  • 元组：tuple
    • 用于存储有序的元素集合不可变序列

2.集合类型

  • 集合：set
    • 集合Set是一种无序且不重复的元素集合，主要用于成员关系测试、去重以及集合运算。
  • 冻结集合：frozenset
    • 不可变集合：使用frozenset()创建不可变集合，元素一旦确定不可修改。

3.映射类型

  • 字典：dict

    • 字典（Dictionary）是一种用于存储键值对的数据结构。字典是无序的（在 Python 3.7+ 中，字典的插入顺序得到了保证），键必须是唯一的且不可变的。
