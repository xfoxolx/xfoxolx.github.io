1.变量及简单数据类型

  • 变量

    • 合法字符
      • 只能包含字母、数字和下划线。
      • 不能以数字开头
      • 区分大小写
      • 不能使用关键字作为变量名
    • 命名习惯
      • 应反应变量的含义
      • 推荐全小写与下划线分割的命名格式
      • 类名的每个单词的首字母都应大写，而不是使用下划线分隔。
    • 私有变量及常量
      • 单下划线开头表示受保护不应直接访问
      • 双下划线开头会触发名称修饰机制用来避免子类覆盖

  • 字符串

    • 字符串创建

      • 单引号或双引号、长端落三引号

    • 访问字符及切片

      • 正向索引从0开始、反向索引从-1开始

    • 字符串拼接与重复

      • 使用 + 运算符或 join() 方法拼接字符串。
      • 使用 * 运算符重复字符串。

    • 字符串常用函数

    • 字符串格式化

      • % 运算符、f 字符串、format() 方法

  • 数字类型

    • 整数类型
      • 没有小数部分、可以为正负 0
      • 无限精度、可表示多进制
    • 浮点数类型
      • 有小数部分、双精度 64 位浮点数存储 -可以使用科学计数法
    • 复数类型
      • 表示复数，形式为 a + bj
    • 类型转换
      • int()、float()、complex()、str()、bool()

  • 布尔类型

    • 布尔值:True、False
    • 布尔类型转换规则
      • 数值：0 转换为 False，非 0 转换为 True
      • 字符串：空字符串为 False，非空字符串为 True
      • 列表、元组、字典、集合：空容器转换为 False，非空容器转换为 True。
    • 布尔值的内置函数
      • bool()、all()、any()

  • None 类型

    • 使用 is 或 is not 运算符来判断一个变量是否为 None，而不是使用 ==
    • None 是 Python 中表示“无值”或“空值”的特殊类型。

  • 运算符

    • 算术运算符
      • +、-、*、/、//、%、**
    • 比较运算符
      • ==、!=、>、<、>=、<=
    • 逻辑运算符
      • and、or、not
    • 成员运算符
      • in、not in
    • 身份运算符
      • is、is not

2.控制结构

  • 条件语句

    • 基本if语句
    • if-else语句
    • if-elif-else语句
    • 嵌套if语句
    • 三元运算符

  • 循环语句

    • for 循环
    • while 循环
      • while条件为True时重复执行代码块。条件为False时，循环结束。
      • while循环通常用于处理未知次数的循环。 故必须有计数器。
    • break 语句
      • break语句用于中断循环。
    • continue 语句
      • continue语句用于跳过当前循环的剩余语句，并继续下一次循环。
    • pass 语句
      • pass语句用于占位符，表示一个空语句，通常用于在需要语句但暂时没有实际代码的情况。
    • else子句
      • else子句用于在正常循环结束后执行一些代码。
    • 循环中的else子句
    • 循环嵌套
      • 外层循环单次执行内部循环全部执行。
